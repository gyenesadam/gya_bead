Backend:
Csatlakozás az adatbázishoz.
Az adatbázisba való adatok felvitele, módosítása, és törlése.

Frontend:

Angular 12.
Az alapvető elképzelés egy youtube-oktató videó alapján megírt adatbázis megjelenítő, valamitn adatkezelő oldal lett volna.
Sajna a verziók közbeszóltak, így ami stabilan működik, az a táblák adatainak kilistázása.
A "Hozzáadás' gombok egy "Modal" felugró ablakot jelenítenének meg, melyekkel lehetőség nyílna a táblák módosítására,
azonban eddig a pontig nem sikerült eljutnom, mert a különböző szoftververziókkal akadtak problémáim, valamint volt néhány ismeretlen hibajelenség is.
Amennyiben szerencsés napom volt, a felugró ablakok megjelentek, de amint továbbhaladtam volna, szétesett az egész, így a határidő előbb eljött, mint a megoldás.
